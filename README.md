# 🎯 Number Guessing Game (Python)

A simple yet robust command-line Number Guessing Game built with Python.  
This project demonstrates core programming concepts such as control flow, user input handling, and exception management.



## 📌 Overview
The application generates a random number between 1 and 100.  
The user attempts to guess it, and after each guess, the program provides feedback to guide the user toward the correct answer.



## ✨ Features
- 🎯 Random number generation using Python's `random` module  
- 🔁 Continuous gameplay until the correct guess  
- 🔢 Attempt counter to track user performance  
- 💡 Intelligent hints ("Too low" / "Too high")  
- ⚠️ Input validation using exception handling (`try-except`)  
- 👶 Beginner-friendly and easy to extend  



## 🛠️ Technologies Used
- Python 3  
- Standard Library: `random`
